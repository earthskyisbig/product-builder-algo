# Dinner Recommendation Generator

## Overview

A simple and visually appealing web application that helps users decide what to have for dinner by generating random meal recommendations. The application will be built with HTML, CSS, and JavaScript, following modern web standards and design principles.

## Features & Design

### Core Functionality
- **Generate Recommendation:** A user can click a button to get a random dinner recommendation.
- **Display Recommendation:** The generated dinner recommendation is displayed clearly on the screen.
- **Theme Switching:** Users can toggle between light and dark modes. The preference will be saved locally.

### Visual Design
- **Theme:** A modern and clean design, adaptable to light and dark modes, with an inviting culinary aesthetic.
- **Layout:** A centered, responsive layout that works well on both desktop and mobile devices.
- **Colors:** An appealing color palette that complements the food theme and adapts for dark mode.
- **Typography:** Clear and readable fonts will be used for all text elements, with an emphasis on readability for the recommendations.
- **Effects:** Subtle shadows and transitions will be used to create a sense of depth and interactivity.
